Aprendendo Funções na Prática

O que é uma Função?

Pense em uma função como uma **máquina de fazer tarefas**. Você entrega ingredientes (dados de entrada), e a máquina te devolve um produto pronto (resultado). Elas servem para organizar o código, evitar repetições e deixar tudo mais claro. Vamos ver como elas funcionam com exemplos do dia a dia!

1. Somando as compras da padaria

Imagine que você comprou um pão e um leite. Esta função soma os dois valores para te dar o total.

Resultado:

Código da Função:

function somar(valor1, valor2) {
  return valor1 + valor2;
}

2. Calculando a área de um terreno

Para saber o tamanho de um terreno retangular, precisamos multiplicar a largura pelo comprimento. É isso que esta função faz.

Resultado:

Código da Função:

function calcularArea(largura, comprimento) {
  return largura * comprimento;
}

3. Verificando se pode dirigir

No Brasil, é preciso ter 18 anos ou mais para tirar a carteira de motorista. Esta função verifica a idade e diz se a pessoa está apta.

Resultado:

Código da Função:

function verificarMaioridade(idade) {
  if (idade >= 18) {
    return "Pode tirar a CNH!";
  } else {
    return "Ainda não pode dirigir.";
  }
}

4. Convertendo Temperatura

Se você vir uma temperatura em Celsius (°C) e quiser saber quanto é em Fahrenheit (°F), use esta função.

Resultado:

Código da Função:

function converterParaFahrenheit(celsius) {
  return (celsius * 9/5) + 32;
}

5. Juntando Nome e Sobrenome

Para preencher um cadastro, você geralmente informa o nome e o sobrenome separados. Esta função junta os dois para criar o nome completo.

Resultado:

Código da Função:

function juntarNomes(nome, sobrenome) {
  return nome + " " + sobrenome;
}

6. Calculando desconto de um produto

Viu um produto em promoção? Esta função calcula o valor final após aplicar um percentual de desconto.

Resultado:

Código da Função:

function calcularPrecoFinal(preco, desconto) {
  const valorDesconto = preco * (desconto / 100);
  return preco - valorDesconto;
}

7. É Par ou Ímpar?

Uma brincadeira clássica. A função usa matemática para descobrir se um número é par ou ímpar.

Resultado:

Código da Função:

function checarParOuImpar(numero) {
  if (numero % 2 === 0) {
    return "É Par";
  } else {
    return "É Ímpar";
  }
}

8. Contador de Caracteres

Útil para redes sociais com limite de texto. A função conta quantos caracteres (letras, espaços, etc.) tem uma frase.

Resultado:

Código da Função:

function contarLetras(texto) {
  return texto.length;
}

9. Calculando Média Escolar

Para passar de ano, você precisa de uma média. Esta função calcula a média de três notas e diz se o aluno foi aprovado.

Resultado:

Código da Função:

function calcularMedia(n1, n2, n3) {
  const media = (n1 + n2 + n3) / 3;
  if (media >= 7) {
    return \`Aprovado com média \${media.toFixed(1)}\`;
  } else {
    return \`Reprovado com média \${media.toFixed(1)}\`;
  }
}

10. Calculando o IMC

O Índice de Massa Corporal (IMC) é uma medida usada para saber se o peso está adequado à altura. A função faz este cálculo.

Resultado:

Código da Função:

function calcularIMC(peso, altura) {
  const imc = peso / Math.pow(altura, 2);
  return \`Seu IMC é \${imc.toFixed(2)}\`;
}
